JavaScript Pro
==============
This class focus on those Javascript skills that are necessary to organize your code into object oriented libraries
and frameworks for reusability.

Topics
------
* Advanced Objects
* Prototype Challenges
* Design Patterns
* ECMA 5 Objects
* TDD with Jasmine
* Building a Game Card
* Async Programming
